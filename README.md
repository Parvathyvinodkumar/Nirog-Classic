# 🩺 NIROG Classic

A healthcare assistance website prototype developed during my first semester of B.Tech. This project was built using HTML, CSS, and JavaScript to explore frontend web development while addressing healthcare accessibility challenges, especially for rural communities.

Although this is a static prototype, it demonstrates the initial concept that later inspired a more advanced full-stack version of NIROG.

---

## 📌 Project Overview

NIROG was designed as a digital healthcare assistance platform that helps users access essential medical services through a simple and user-friendly interface.

The project focuses on:

- Emergency healthcare assistance
- Hospital discovery
- Ambulance request simulation
- First-aid guidance
- Basic healthcare information
- Administrator portal
- Doctor and hospital information pages

---

## ✨ Features

- 👤 User Login Interface
- 🔐 Administrator Login
- 🚑 Ambulance Request Module
- 📍 Hospital Finder using Browser Geolocation
- 🏥 Hospital Information Pages
- 👨‍⚕️ Doctor Information Pages
- 🩹 Emergency First Aid Guide
- 💬 Basic Healthcare Chatbot Interface
- 📱 Responsive UI Design

## 🎯 Motivation

The objective of this project was to understand frontend web development while designing a healthcare platform that could potentially assist users in locating hospitals, accessing emergency information, and requesting medical assistance.

This project served as my introduction to software development and user interface design.
